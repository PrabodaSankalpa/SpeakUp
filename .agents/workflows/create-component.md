# Workflow: Create Component

1. **Check**: Ensure the component name is unique in `src/components/`.
2. **Template**:
   - Import React hooks.
   - Export a functional component.
   - Include Tailwind classes for basic styling.
   - Add `PropTypes` or basic JSDoc for parameters.
3. **Register**: Mention the new component in `App.jsx` if it’s a primary view.
